群星(Stellaris)因校验码不同导致无法联机的解决方法
发表于2021-05-29|教程|游戏•Stellaris
几天前和朋友们联机群星，有位小伙伴因为网络不稳定中途掉线了，等他再尝试加入时，一直被提示“校验码不匹配”，但他没有修改任何游戏文件。据他说，之前他和别人联机时，和他联机的人也遇到过这种情况，看来是老Bug了，P社也一直没管QAQ。 网上找到的教程大多都是要你验证游戏完整性，但这很费时，而且我的小伙伴在验证完后也还是进不来xD。甚至卸了群星再重装后，校验码也还是不对;w; 翻遍了网上的各种教程，最后发明了这个省时省力的方法，成功解决了问题 找到并了解checksum_manifest.txt 在steam左栏中右键群星→\rightarrow→ 属性→\rightarrow→本地文件→\rightarrow→浏览来打开游戏目录 在游戏目录下找到这个叫checksum_manifest的文本，打开它 这个文本里存的就是群星在计算校验码时读取的文件列表了 选文本中一部分稍微解释一下 1234directory name = common sub_directories = yes file_extension = .txt name=common：读取游戏目录下common中的文 ...
